Auth / usage notes

All protected endpoints require Authorization: Bearer <access_token> returned by POST /api/auth/login.
Roles (from core/permissions.py): admin, senior_manager, hr_recruiter, employee. Routes are role-protected.

How I suggest you use this

Use the file links above to inspect the exact request/response Pydantic schemas. Example: ResumeScreenResponse is in backend/schemas/ai.py.
Seed your frontend API client to call these endpoints and use the example JSON above as mock fixtures while building UI.
Authorization: add global axios/Fetch wrapper that injects Authorization: Bearer <token>. Use the role in the token payload to conditionally show dashboards.
